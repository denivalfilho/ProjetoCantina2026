Projeto Cantina Fatec

Sistema de gerenciamento de estoque e vendas desenvolvido para as disciplinas de Estrutura de Dados e Linguagem de Programação 2.
Funcionalidades

Gerenciamento de estoque com cadastro manual e baixa automática em vendas.
Geração de dados com a biblioteca Faker para criar massa de testes de alunos e produtos.
Persistência de dados com uso de Pickle para salvamento em arquivo binário .dat.
Relatórios de extrato financeiro, consumo por cliente e análise de reposição.
Conceitos Aplicados

Estrutura de Dados com implementação manual de Lista Encadeada Simples.
Programação Orientada a Objetos com encapsulamento, atributos privados e métodos de acesso.
Persistência com serialização de objetos para manutenção de dados não voláteis.
Execução

Instalar dependências: pip install faker

Iniciar o sistema: python main.py
