Projeto Cantina Fatec

Trabalho de Estrutura de Dados e LP2. Sistema para controlar estoque e vendas da cantina da faculdade.
O que tem no projeto:

    Lista Encadeada: Fiz na mão usando classes e o ponteiro __proximo.

    Encapsulamento: Todos os atributos são privados (__) com Getters e Setters.

    Faker: Usei para gerar nomes de alunos e vendas de teste.

    Pickle: Salva os dados num arquivo .dat para não perder nada quando fecha.

    Relatórios: Mostra as vendas, quem comprou e o que precisa repor.

Como rodar:

    pip install faker

    python main.py
